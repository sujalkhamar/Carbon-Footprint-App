# Carbon Footprint App

A simple Android application that helps users understand their carbon footprint by entering daily activity data such as travel distance and electricity usage.

## Features

- Calculate carbon footprint based on user input
- Accept travel distance details
- Accept electricity usage details
- Show estimated carbon impact
- Simple and user-friendly Android interface

## Tech Stack

- Java
- Android Studio
- XML Layouts

## Project Structure

- `app/src/main/java` - Java source code
- `app/src/main/res/layout` - XML layout files

## How It Works

The app collects user activity data such as:
- Travel distance
- Electricity usage

Using this input, it estimates the environmental impact and helps users become more aware of their carbon emissions.

## Example Inputs

- Travel distance: `15 km`
- Electricity usage: `5 kWh`
